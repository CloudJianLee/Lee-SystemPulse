# System Pulse 1.0.0

首个正式版本现已发布。

## 主要功能

- 在 macOS 菜单栏实时显示 CPU 与内存使用率
- CPU 进度从左向右增长，内存进度从右向左增长
- 百分比统一使用白色显示，兼顾深色与浅色菜单栏
- 负载颜色分级：
  - 0%–79%：充电绿
  - 80%–89%：警告橙
  - 90%–94%：警告红
  - 95%–100%：深红严重拥塞
- 支持开机自动启动
- 中文状态面板与内存压力提示
- 原生 SwiftUI 实现，支持 Apple Silicon 与 Intel Mac

## 安装

下载 `System-Pulse-1.0.0.dmg`，打开后将“System Pulse”拖入“应用程序”文件夹。

当前安装包使用临时签名，尚未经过 Apple 公证。首次启动时，请在 Finder 中右键应用并选择“打开”，然后再次确认。

可使用同目录下的 `.sha256` 文件核对安装包完整性。
